# Ceratops CSS

CSS framework designed to use for every web project.

## Install

### Yarn

```sh
yarn add ceratops
```

or

### NPM

```sh
npm install ceratops
```

## Usage

### CSS

```css
@import 'ceratops/css/ceratops.min.css';
```

or

### HTML

```html
<link rel="stylesheet" href="ceratops/css/ceratops.min.css" />
```
